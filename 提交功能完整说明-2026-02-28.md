## 🎯 功能概述

当你点击"提交表单"按钮时，系统会自动完成以下操作：

1. ✅ **验证表单** - 检查所有必填字段
2. ✅ **保存数据** - 将表单数据保存到数据库
3. ✅ **上传附件** - 自动上传所有内联附件（16个字段）
4. ✅ **提交锁定** - 提交到考评小组并锁定表单

**重要：表单和附件会一起自动提交，你不需要做任何额外操作！**

---

## 📋 提交流程详解

### 第 1 步：验证表单
```
检查项：
- 所有必填字段是否已填写
- 自评分是否在有效范围内（0-10分）
- 数据格式是否正确
```

### 第 2 步：保存自评表
```
操作：
- 将表单数据发送到后端
- 后端保存到数据库
- 返回 evaluation_id
```

### 第 3 步：上传附件
```
自动上传以下字段的附件（如果有）：

常规教学工作（8个）：
1. 教学过程管理
2. 教学质量管理
3. 课程考核
4. 教育教学科研工作
5. 课程建设
6. 教师队伍建设
7. 科学研究与学术交流
8. 教学档案室管理与建设

特色与亮点项目（4个）：
1. 教学改革项目
2. 年度获得教学相关荣誉表彰
3. 教学比赛
4. 指导创新创业比赛获奖情况

负面清单（4个）：
1. 教师中有严重违反师德师风的行为
2. 出现教学（管理）事故
3. 教学过程中出现意识形态问题
4. 教师未完成年度基本教学工作量

总计：16 个字段的附件会自动上传
```

### 第 4 步：提交到考评小组
```
操作：
- 锁定表单，不可再修改
- 将数据提交到考评小组端
- 考评小组可以开始评分
```

### 父组件（TeachingOfficeHome.vue）

```typescript
const handleSubmit = async (submitData: any) => {
  const { formData, attachments, uploadAttachments } = submitData
  
  // 步骤 1: 保存自评表
  const saveResponse = await selfEvaluationApi.save({
    teaching_office_id: authStore.teachingOfficeId,
    evaluation_year: new Date().getFullYear(),
    content: formData
  })
  
  const evaluationId = saveResponse.data.evaluation_id
  
  // 步骤 2: 上传附件（如果有）
  if (attachments && attachments.length > 0) {
    await uploadAttachments(evaluationId)
  }
  
  // 步骤 3: 提交到考评小组
  await selfEvaluationApi.submit(evaluationId)
  
  ElMessage.success('提交成功！数据已上传到考评小组端')
}
```

## 🔍 常见问题

### Q1: 我需要先上传附件再提交表单吗？
**A**: 不需要！只需在字段右侧选择附件，点击"提交表单"时会自动上传。

### Q2: 如果附件上传失败怎么办？
**A**: 表单数据仍会保存，系统会提示"附件上传失败，但表单已保存"，你可以稍后单独上传附件。

### Q3: 提交后还能修改吗？
**A**: 不能。提交后表单会被锁定，按钮变为"已提交"状态。

### Q4: 可以不上传附件直接提交吗？
**A**: 可以！附件是可选的，你可以只提交表单数据。

### Q5: 提交需要多长时间？
**A**: 
- 无附件：约 1-2 秒
- 有附件：取决于附件大小和数量，通常 5-30 秒
