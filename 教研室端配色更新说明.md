## 🎓 设计理念

### 为什么选择藏蓝色？

#### 1. 符合教研室定位
- **学术性**: 藏蓝色是传统学术机构的代表色
- **专业性**: 深沉的蓝色体现教育的专业性
- **权威性**: 深色调体现教研室的权威地位
- **稳重性**: 适合教育工作的严谨氛围

#### 2. 教育行业标准色
- **世界名校**: 哈佛、耶鲁、牛津等名校都使用深蓝色
- **学术期刊**: 许多学术期刊封面使用藏蓝色
- **教育机构**: 教育部门常用深蓝色作为主色调
- **学位服**: 博士学位服常用藏蓝色

#### 3. 心理学效果
- **蓝色**: 代表智慧、知识、冷静、理性
- **深蓝**: 代表深度、专业、权威、信任
- **藏蓝**: 结合两者，既有学术深度又有专业权威

#### 4. 与其他端口区分
- **教研室端**: 藏蓝色 (#1a237e) - 学术、专业
- **管理端**: 紫蓝色 (#667eea) - 权威、科技
- **数据大屏**: 深蓝黑 (#0a0e27) - 科技、炫酷

|------|--------|-----|------|------|---------|

```
深度递增: 数据大屏 > 教研室端 > 管理端
明度递增: 数据大屏 < 教研室端 < 管理端
饱和度: 教研室端 > 管理端 > 数据大屏
```

## 🔍 细节优化

### 1. 渐变设计
- **从单色到渐变**: 增加视觉层次
- **渐变方向**: 135度斜向，增加动感
- **色彩过渡**: 从藏蓝到深蓝，自然流畅
- **视觉效果**: 立体感、专业感、学术氛围

### 2. 透明度处理
- 侧边栏菜单悬停: `rgba(255, 255, 255, 0.05)`
- 侧边栏菜单激活: `rgba(255, 255, 255, 0.1)`
- 用户信息背景: `rgba(255, 255, 255, 0.05)`
- 保持与藏蓝色背景的协调

### 3. 阴影效果
- 功能卡片悬停阴影: `rgba(26, 35, 126, 0.15)`
- 使用藏蓝色的半透明版本
- 增强悬停反馈
- 保持色彩一致性

### 4. 角色专属配色（保持不变）
- 普通教师: #42a5f5 (天蓝色)
- 教研室负责人: #66bb6a (深绿色)
- 二级学院负责人: #ab47bc (紫色)
