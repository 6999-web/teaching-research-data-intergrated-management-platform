# 评分感悟改为评分原因

## 修改内容

将手动评分页面中的"评分感悟"改为"评分原因"。

## 修改位置

文件：`frontend/src/components/ManualScoringForm.vue`

### 修改1：表单标签

**修改前：**
```vue
<el-form-item 
  label="评分感悟"
  :prop="`scores.${itemKey}.comment`"
>
  <el-input
    placeholder="请填写评分说明和感悟"
  />
</el-form-item>
```

**修改后：**
```vue
<el-form-item 
  label="评分原因"
  :prop="`scores.${itemKey}.comment`"
>
  <el-input
    placeholder="请填写评分原因和说明"
  />
</el-form-item>
```

### 修改2：错误提示信息

**修改前：**
```typescript
ElMessage.error(`请填写「${label}」的评分感悟（至少2个字符）`)
```

**修改后：**
```typescript
ElMessage.error(`请填写「${label}」的评分原因（至少2个字符）`)
```

## 影响范围

- ✅ 手动评分表单的标签文本
- ✅ 输入框的占位符文本
- ✅ 表单验证的错误提示信息
