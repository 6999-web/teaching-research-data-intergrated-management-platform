| 课程建设 | 每门课程配备主讲教师，所有课程均有规范的教学大纲、教案和课件，课程定位准确，目标清晰合理，教学内容安排合理，能够有机融入课程思政元素，体现以学生发展为中心，能有效支撑课程目标。 |     |     |
| 教师队伍建设 | 制订有教师培养规划，有计划地安排青年教师的教学、教研和进修深造工作 |     |     |
| 科学研究与学术交流 | 承担有科研项目，开展有计划、有组织的社会服务活动，参与校内外、国内外的学术交流活动，为学生举办学术讲座。 |     |     |
| 教学档案室管理与建设 | 各类教学档案齐全，教学管理档案归档及时。 |     |     |
