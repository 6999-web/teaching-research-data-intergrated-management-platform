【重要说明】

本系统采用 **统一平台架构**，三个端口共享同一个前端和后端服务：

✅ 前端服务 (端口 3000) - 统一入口
✅ 后端服务 (端口 8000) - 统一API

三个"端口"是通过前端左侧导航栏切换的不同功能模块：
- 教研室端
- 管理端  
- 校长办公会端

【三端切换方式】

访问 http://localhost:3000 后：

1️⃣ 教研室端
   点击左侧导航栏 "教研室端"

   ├─ 详教方案
   │  ├─ 填写自评表

2️⃣ 管理端
   点击左侧导航栏 "管理端"
   
   功能模块:

   │  ├─ 手动评分
   │  ├─ 最终得分
   │  └─ 异常处理
   ├─ 分析中心
   │  ├─ 数据同步
   │  └─ 发起公示
   └─ 应用中心
      ├─ 附件管理
      └─ 系统配置

3️⃣ 校长办公会端
   点击左侧导航栏 "校长办公会端"
   
   功能模块:
   ├─ 数据看板
   │  ├─ 实时监控
   │  └─ 数据分析
   └─ 审批中心
      ├─ 结果审定
      └─ 公示管理

【使用流程】

1. 访问 http://localhost:3000
2. 在左侧导航栏选择端口（教研室端/管理端/校长办公会端）
3. 在顶部标签栏选择功能模块
4. 点击功能卡片进入具体页面
5. 点击左上角返回主页

【技术特点】

🎯 单页应用 (SPA)
   - 前端路由切换
   - 无需刷新页面
   - 流畅的用户体验

🔄 前后端分离
   - 前端: Vue 3 + TypeScript
   - 后端: FastAPI + Python
   - RESTful API 通信

🎨 统一设计
   - 警察蓝色配色
   - 平台化布局
   - 响应式设计

🔐 权限控制
   - 基于角色的访问控制
   - JWT Token 认证
   - API级别权限验证

管理端:
- POST /api/scoring/manual-score

- GET  /api/review/anomalies
- POST /api/publication/publish

校长办公会端:
