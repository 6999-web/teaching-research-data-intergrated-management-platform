# 教研室端功能优化说明

## 📋 更新概述

**更新时间**: 2026-02-09  
**更新内容**: 优化教研室端功能结构  
**影响范围**: 教研室端主页、登录页面、状态管理、流程条组件

---

## ✨ 主要变更

### 1. 删除普通教师角色 ❌
- **原因**: 简化角色结构，聚焦核心用户
- **影响**: 
  - 登录页面不再显示"普通教师"选项
  - 教研室端只保留2个角色：
    - 教研室负责人
    - 二级学院负责人

### 2. 删除AI评分菜单 ❌
- **原因**: AI评分由系统自动触发，无需手动操作
- **影响**: 
  - 教研室负责人菜单从4项减少到3项
  - 删除"AI评分"菜单项
  - 删除相关功能卡片

### 3. 重命名功能 ✏️

| 旧名称 | 新名称 | 说明 |
|--------|--------|------|
| 自评管理 | 填写自评表 | 更明确的功能描述 |
| 材料管理 | 材料提交 | 强调提交动作 |
| 改进闭环 | 改进措施 | 简化名称 |

### 4. 优化标签页 📑

#### 教研室负责人标签页
- **填写自评表**: 教研室工作考核评分表
- **材料提交**: 附件列表
- **改进措施**: 改进措施表

### 5. 优化功能卡片 🎴

#### 填写自评表
- 填写自评表：在线填写教研室工作考核评分表
- 查看自评表：查看已提交的自评表内容

#### 材料提交
- 上传附件：上传支撑材料，支持多文件批量上传
- 附件列表：查看和管理所有附件

#### 改进措施
- 填写改进措施：针对薄弱项填写改进措施
- 查看审核状态：查看改进措施的审核状态

## 🔄 流程优化

### 教研室负责人工作流程

#### 旧流程
```
1. 登录系统
2. 选择"自评管理"
3. 填写自评表
4. 选择"材料管理"
5. 上传附件
6. 选择"AI评分"
7. 触发AI评分
8. 等待评分结果
9. 选择"改进闭环"
10. 填写改进措施
```

#### 新流程
```
1. 登录系统
2. 选择"填写自评表"
3. 填写教研室工作考核评分表
4. 选择"材料提交"
5. 上传附件
6. 系统自动触发AI评分
7. 选择"改进措施"
8. 填写改进措施表
```

**优化点**:
- ✅ 减少菜单层级
- ✅ 简化操作步骤
- ✅ AI评分自动化
- ✅ 名称更直观

## 🚀 使用指南

### 教研室负责人登录

1. **访问系统**: http://localhost:3000
2. **点击"教研室端"卡片**
3. **输入用户名和密码**
4. **选择角色**: 教研室负责人
5. **点击"登录"**

### 填写自评表

1. **点击"填写自评表"菜单**
2. **查看"教研室工作考核评分表"标签**
3. **点击"填写自评表"卡片**
4. **在线填写表单**
5. **保存并提交**

### 材料提交

1. **点击"材料提交"菜单**
2. **查看"附件列表"标签**
3. **点击"上传附件"卡片**
4. **选择文件上传**
5. **查看附件列表**

### 改进措施

1. **点击"改进措施"菜单**
2. **查看"改进措施表"标签**
3. **点击"填写改进措施"卡片**
4. **填写改进措施**
5. **查看审核状态**
