# 双模式使用指南

## 🎯 什么是双模式？

教研室数据管理平台现在支持两种工作模式：

1. **三端模式**（原有）：按功能端口划分，适合管理员和熟练用户
2. **四角色模式**（新增）：按用户角色划分，适合新用户和需要流程指引的用户

您可以随时在两种模式之间切换，系统会记住您的选择。

---

## 🚀 快速开始

### 第一步：打开系统

1. 打开浏览器
2. 访问 http://localhost:3000
3. 系统默认显示**三端模式**

### 第二步：查看当前模式

查看左侧导航栏顶部：

```
┌────────────────────┐
│  [校徽图片]        │
│  教研室数据管理平台 │
│  [当前端口/角色]   │  ← 这里显示当前模式
│                    │
│  [切换按钮]        │  ← 点击这里切换模式
└────────────────────┘
```

### 第三步：切换模式

点击切换按钮即可切换模式：

- 如果当前是三端模式，按钮显示"切换到四角色模式"
- 如果当前是四角色模式，按钮显示"切换到三端模式"

### 三个端口

#### 1. 教研室端

**适用人员**: 教研室负责人、教师

**主要功能**:
- 📝 详教方案
  - 填写自评表
  - 查看结果
- 📎 分发材料
  - 上传附件
  - 附件管理

**使用场景**: 填写自评表、上传材料、查看考评结果

#### 2. 管理端

**适用人员**: 考评小组、考评办公室

**主要功能**:
- 📊 监控中心
  - 手动评分
  - 最终得分
  - 异常处理
- 📈 分析中心
  - 数据同步
  - 发起公示
- ⚙️ 应用中心
  - 附件管理
  - 系统配置

**使用场景**: 评分、处理异常、发起公示

#### 3. 校长办公会端

**适用人员**: 校长办公会成员

**主要功能**:
- 📊 数据看板
  - 实时监控
  - 数据分析
- ✅ 审批中心
  - 结果审定
  - 公示管理

**使用场景**: 查看全校数据、审定结果

### 操作步骤

1. **选择端口**: 点击左侧导航栏的端口名称
2. **选择标签**: 点击顶部标签栏的标签
3. **选择功能**: 点击功能卡片进入对应页面
4. **执行操作**: 在功能页面中完成具体操作


### 四个角色

#### 1. 普通教师 👨‍🏫

**权限**: 只读权限

**主要功能**:
- 📊 我的教研室
  - 考评结果
  - 评分详情
  - 改进措施
- 📈 数据分析
  - 历史对比
  - 指标雷达图

**流程节点**: 高亮节点 1, 2, 8

**使用场景**: 查看本教研室的考评结果和改进措施

**操作提示**:
- ✅ 可以查看考评结果和详细评分
- ✅ 可以查看改进措施和执行进度
- ❌ 不能编辑或修改任何数据

#### 2. 教研室负责人 👨‍💼

**权限**: 读写权限（仅限本教研室）

**主要功能**:
- 📝 自评管理
  - 填写自评表
  - 查看自评表
- 📎 材料管理
  - 上传附件
  - 附件列表
- 🤖 AI评分
  - 触发评分
  - 评分结果
- 🔄 改进闭环
  - 提交改进措施
  - 查看审核状态

**流程节点**: 高亮节点 1, 2, 3, 8

**使用场景**: 填写自评表、上传材料、触发AI评分、提交改进措施

**操作提示**:
- ✅ 负责填写和提交自评表
- ✅ 负责上传支撑材料
- ✅ 可以触发AI自动评分
- ✅ 需要提交改进措施并等待学院审核

#### 3. 二级学院负责人 👨‍🎓

**权限**: 只读权限（全院）+ 审核权限

**主要功能**:
- 📊 学院驾驶舱
  - 全院概览
  - 教研室对比
  - 指标分析
- 🔍 整改督导
  - 改进措施审核
  - 执行进度跟踪
  - 督导记录

**流程节点**: 高亮节点 8

**使用场景**: 查看全院数据、审核改进措施、督导整改落实

**操作提示**:
- ✅ 可以查看全院所有教研室的数据
- ✅ 负责审核教研室提交的改进措施
- ✅ 可以添加督导批注
- ✅ 跟踪改进措施的执行进度

#### 4. 学校教务处负责人 👨‍💻

**权限**: 读写权限（全校）+ 管理权限

**主要功能**:
- 📊 监控中心
  - 全校概览
  - 实时数据
  - AI评分监控
- ✍️ 评分管理
  - 手动评分
  - 最终得分确定
  - 评分记录查询
- ⚠️ 异常处理
  - 异常数据列表
  - 处理记录
- 📢 公示管理
  - 发起公示
  - 公示状态
  - 数据同步

**流程节点**: 高亮节点 3, 4, 5, 7

**使用场景**: 手动评分、确定最终得分、处理异常、发起公示

**操作提示**:
- ✅ 可以查看全校所有数据
- ✅ 负责手动评分和确定最终得分
- ✅ 处理AI标记的异常数据
- ✅ 发起考评结果公示

### 流程条说明

流程条显示整个考评流程的8个步骤：

```
1. 自评填报 → 2. 材料上传 → 3. AI评分 → 4. 手动评分 
→ 5. 最终得分 → 6. 数据审定 → 7. 结果公示 → 8. 改进闭环
```

**颜色说明**:
- 🟢 绿色：已完成的步骤
- 🔵 蓝色：当前步骤
- ⚪ 灰色：未开始的步骤
- 🟠 橙色边框：当前角色负责的步骤

### 操作步骤

1. **选择角色**: 点击左侧导航栏的角色名称
2. **查看流程**: 观察流程条，了解当前所处的流程节点
3. **选择标签**: 点击顶部标签栏的标签
4. **选择功能**: 点击功能卡片进入对应页面
5. **执行操作**: 在功能页面中完成具体操作

## 🔄 模式切换技巧

### 何时使用三端模式？

✅ **推荐使用场景**:
- 您是系统管理员，需要访问所有功能
- 您熟悉系统功能结构，按功能模块操作更高效
- 您需要快速在不同功能模块之间切换
- 您不需要严格的角色权限控制

### 何时使用四角色模式？

✅ **推荐使用场景**:
- 您是新用户，首次使用系统
- 您有明确的角色定位（教师、负责人、学院领导、教务处）
- 您需要清晰的工作流程指引
- 您需要了解当前在整个流程中的位置
- 您需要严格的角色权限控制

### 快速切换

1. **一键切换**: 点击侧边栏顶部的切换按钮
2. **自动保存**: 系统会记住您的选择
3. **自动恢复**: 下次打开系统时自动恢复上次的模式

## 💡 使用技巧

### 技巧 1: 点击标题返回首页

点击侧边栏顶部的校徽或平台名称，可以快速返回首页（重置到默认模块）。

### 技巧 2: 查看角色徽章

侧边栏顶部的角色徽章会显示当前的端口或角色名称，帮助您快速确认当前位置。

### 技巧 3: 利用流程条

在四角色模式下，流程条会高亮显示当前角色负责的步骤，帮助您了解工作职责。

### 技巧 4: 功能卡片悬停

将鼠标悬停在功能卡片上，卡片会有动画效果，并显示箭头图标，提示可以点击。

### 技巧 5: 标签页切换

顶部标签栏可以快速切换不同的功能分类，无需返回上一级。

## 🎯 典型工作流程

### 流程 1: 教研室负责人填写自评表（四角色模式）

4. 触发AI自动评分
5. 流程条显示完成"AI评分"步骤

1. 切换到"学校教务处负责人"角色

3. 点击"手动评分"功能卡片

5. 流程条显示完成"手动评分"步骤

2. 选择"管理端"

## ❓ 常见问题

### Q1: 如何知道当前是哪种模式？

**A**: 查看侧边栏顶部的切换按钮文字：
- 如果显示"切换到四角色模式"，说明当前是三端模式
- 如果显示"切换到三端模式"，说明当前是四角色模式

### Q2: 切换模式后数据会丢失吗？

**A**: 不会。切换模式只是改变界面布局和功能组织方式，不会影响数据。

### Q3: 刷新页面后模式会重置吗？

**A**: 不会。系统会记住您的模式选择，刷新页面后自动恢复。

### Q4: 为什么三端模式没有流程条？

**A**: 三端模式按功能模块组织，不需要流程条。流程条只在四角色模式下显示，用于展示工作流程。

### Q5: 可以同时使用两种模式吗？

**A**: 不可以。同一时间只能使用一种模式，但您可以随时切换。

### Q6: 不同角色看到的功能一样吗？

**A**: 不一样。在四角色模式下，不同角色看到的功能和权限都不同。

### Q7: 如何快速返回首页？

**A**: 点击侧边栏顶部的校徽或平台名称即可返回首页。

### Q8: 移动端可以使用吗？

**A**: 可以。系统支持响应式设计，在移动端会自动调整布局。
