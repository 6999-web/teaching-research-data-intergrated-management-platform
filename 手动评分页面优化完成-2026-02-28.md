### 1. 时间显示格式修改 ✓

- 为每个评分项添加独立的评分和评语字段
- 评分区域包括：
  - 评分输入框（数字输入，带最大值限制）
  - 评分感悟输入框（多行文本，500字限制）
- 使用响应式布局：
  - 桌面端：内容区域 2fr，评分区域 1fr

- 删除了原有的三个标签页：
  - 自评表内容
  - 附件材料
  - 手动评分
- 改为单一整合视图，所有内容在一个页面显示
- 简化了用户操作流程

## 技术实现

### 数据结构
```typescript
// 表单数据结构
formData: {
  scores: {
    [itemKey: string]: {
      score: number
      comment: string
    }
  }
}
```

### 关键方法

1. **initializeFormData()**
   - 遍历自评表内容，为每一项初始化评分数据
   - 在加载自评表内容后自动调用

2. **getAttachmentsForIndicator(indicatorKey)**
   - 根据指标键值筛选相关附件
   - 使用映射表将前端字段名映射到后端指标名

3. **formatDateTime(dateStr)**
   - 格式化时间为 `YYYY-MM-DD HH:mm:ss`
   - 手动拼接年月日时分秒

### 布局结构
```
.content-item-with-scoring
├── .item-content-area (2fr)
│   ├── .item-header (标题 + 自评分)
│   ├── .item-content (内容)
│   └── .item-attachments (附件列表)
└── .scoring-area (1fr)
    ├── 评分输入框
    └── 评分感悟输入框
```
