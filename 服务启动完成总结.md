**后端配置** (`backend/.env`):
```env
MYSQL_SERVER=localhost
MYSQL_USER=root
MYSQL_PASSWORD=root
MYSQL_DB=teaching_office_evaluation
MYSQL_PORT=3306
```

**前端配置** (`frontend/.env.development`):
```env
VITE_API_BASE_URL=/api
```

**Vite代理配置** (`frontend/vite.config.ts`):
```typescript
proxy: {
  '/api': {
    target: 'http://localhost:8000',
    changeOrigin: true
  }
}
```

### 端口使用

- **3000**: 前端服务
- **8000**: 后端服务
- **3306**: MySQL数据库

确保这些端口没有被其他程序占用。

## 🐛 故障排查

### 问题1: 无法访问前端
**症状**: 浏览器显示"无法访问此网站"

**解决方案**:
1. 检查前端进程是否运行: `listProcesses`
2. 查看前端日志: `getProcessOutput(processId: 4)`
3. 确认端口3000未被占用
4. 重启前端服务

### 问题2: API返回404
**症状**: API请求返回404错误

**解决方案**:
1. 检查API路径是否正确
2. 后端API文档在 `/docs` 而不是 `/api/docs`
3. 查看后端日志确认路由配置

### 问题3: 数据库连接失败
**症状**: 后端启动时报数据库错误

**解决方案**:
1. 检查MySQL服务: `mysql -u root -proot -e "SELECT 1;"`
2. 检查数据库配置: `backend/.env`
3. 确认数据库存在: `teaching_office_evaluation`
