## 🌐 访问地址

```
http://localhost:3000
```

## 🎯 快速测试步骤

### 1. 打开前端页面

在浏览器中访问:
```
http://localhost:3000
```

你应该看到登录页面，有三个入口：
- 教研室端
- 管理端
- 数据大屏

### 2. 测试API文档

在浏览器中访问:
```
http://localhost:8000/docs
```

你应该看到Swagger UI界面，可以测试所有API端点。

### 3. 测试新评分表功能

#### 方法1: 通过前端界面
1. 登录教研室账号
2. 进入"填写考评表"
3. 填写新评分表
4. 提交并查看AI评分

#### 方法2: 通过API文档
1. 访问 http://localhost:8000/docs
2. 找到 `/api/v1/self-evaluations` 端点
3. 点击"Try it out"
4. 输入测试数据
5. 点击"Execute"

1. **首次加载**: 可能需要3-5秒

4. **文件上传**: 取决于文件大小和网络速度

## 🐛 常见问题

### Q: 无法访问前端页面？
A: 检查前端服务是否运行，访问 http://localhost:3000

### Q: API返回401错误？
A: 需要先登录获取token，或在API文档中点击"Authorize"

### Q: 数据库连接失败？
A: 检查MySQL服务是否运行，用户名密码是否正确

### Q: AI评分失败？
A: 检查DeepSeek API密钥是否配置，或使用模拟数据测试
