## 🔧 更新的逻辑

### 1. 提交按钮文本
**修改前**:
```vue
{{ isSubmitted ? '已提交' : (attachmentFileList.length > 0 ? '提交表单并上传附件' : '提交表单') }}
```

**修改后**:
```vue
{{ isSubmitted ? '已提交' : '提交表单' }}
```

### 2. handleSubmit 函数
**修改前**:
```typescript
const allAttachments = [
  ...attachmentFileList.value,  // 底部上传的附件
  ...teachingProcessFiles.value,
  // ... 其他内联附件
]
```

**修改后**:
```typescript
const allAttachments = [
  ...teachingProcessFiles.value,
  ...teachingQualityFiles.value,
  // ... 只包含内联附件
]
```

### 3. uploadAttachments 函数
**修改前**:
```typescript
if (attachmentFileList.value.length === 0) {
  return true
}
// 只上传底部附件
for (const file of attachmentFileList.value) {
  // ...
}
```

**修改后**:
```typescript
// 合并所有内联上传的附件
const allAttachments = [
  ...teachingProcessFiles.value,
  ...teachingQualityFiles.value,
  // ... 所有内联附件
]

if (allAttachments.length === 0) {
  return true
}

// 上传所有内联附件
for (const file of allAttachments) {
  // ...
}
```
