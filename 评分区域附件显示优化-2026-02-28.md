# 评分区域附件显示优化

## 修改内容

### 问题描述
用户希望在考评小组端的手动评分页面中，在评分区域内显示附件，而不是只在左侧内容区域显示。

### 修改方案
在评分区域内，按照以下顺序显示：
1. 评分项名称（标题）
2. 相关附件列表（紧凑版）
3. 评分输入框
4. 评分感悟输入框

### 修改文件
`frontend/src/components/ManualScoringForm.vue`

## 修改详情

### 1. 模板结构调整

**修改前**：
```vue
<div class="scoring-area">
  <el-form>
    <el-form-item label="评分">
      <el-input-number />
    </el-form-item>
    <el-form-item label="评分感悟">
      <el-input />
    </el-form-item>
  </el-form>
</div>
```

**修改后**：
```vue
<div class="scoring-area">
  <!-- 评分项标题 -->
  <div class="scoring-header">
    <h4 class="scoring-title">{{ getItemLabel(itemKey) }}</h4>
  </div>
  
  <!-- 附件显示（紧凑版） -->
  <div v-if="getAttachmentsForIndicator(itemKey).length > 0" class="scoring-attachments">
    <div class="attachments-header-small">
      <el-icon><Paperclip /></el-icon>
      <span>附件 ({{ getAttachmentsForIndicator(itemKey).length }})</span>
    </div>
    <div class="attachment-list-compact">
      <div v-for="attachment in getAttachmentsForIndicator(itemKey)" 
           :key="attachment.id"
           class="attachment-item-compact">
        <el-icon class="file-icon-small"><Document /></el-icon>
        <span class="file-name-small">{{ attachment.file_name }}</span>
        <el-button type="primary" size="small" link 
                   @click="handleDownloadAttachment(attachment)">
          下载
        </el-button>
      </div>
    </div>
  </div>
  
  <!-- 评分表单 -->
  <el-form>
    <el-form-item label="评分">
      <el-input-number />
    </el-form-item>
    <el-form-item label="评分感悟">
      <el-input />
    </el-form-item>
  </el-form>
</div>
```

## 优势

### 1. 更好的信息组织
- 评分项名称作为标题，清晰标识当前评分项
- 附件紧跟在名称后面，方便查看相关材料
- 评分输入在附件下方，符合评分流程

### 2. 紧凑的布局
- 使用紧凑版附件样式，节省空间
- 字体和间距适当缩小，不影响可读性
- 保持整体布局的平衡

### 3. 一致的用户体验
- 左侧内容区域仍然显示完整的附件信息
- 右侧评分区域显示紧凑版附件
- 两处附件显示互为补充

## 注意事项

### 1. 附件显示逻辑
- 只有当该评分项有附件时才显示附件区域
- 使用 `v-if` 条件渲染，避免空白区域

### 2. 样式兼容性
- 紧凑版样式不影响原有的附件显示
- 响应式设计，适配不同屏幕尺寸

### 3. 性能考虑
- 附件列表使用虚拟滚动（如果附件很多）
- 图标使用 Element Plus 内置图标，加载快
