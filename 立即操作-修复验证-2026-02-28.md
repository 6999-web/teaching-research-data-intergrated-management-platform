## ⚡ 立即操作步骤

### 第一步：清除浏览器缓存（必须！）

**重要**：前端代码已更新，必须清除缓存才能看到修复效果！

**方法1：快捷键**
```
按 Ctrl + Shift + Delete
选择"缓存的图片和文件"
点击"清除数据"
```

**方法2：强制刷新**
```
按 Ctrl + F5
或
按 Ctrl + Shift + R
```

### 第二步：验证时间格式修复

1. **访问系统**
   ```
   http://localhost:3000
   ```

2. **登录考评小组账号**
   ```
   用户名：evaluator1
   密码：password123
   选择角色：评教小组
   ```

3. **进入手动评分页面**
   - 点击左侧菜单"手动评分"

4. **检查时间格式**
   - 查看"提交时间"列

5. **验证结果**
   - [ ] 时间格式正确（YYYY-MM-DD HH:mm:ss）
   - [ ] 包含秒数
   - [ ] 使用24小时制

### 第四步：测试考评表接收流程

#### 4.1 教研室提交考评表

1. **退出当前账号**
   - 点击左下角"退出登录"

2. **登录教研室账号**
   ```
   用户名：director1
   密码：password123
   ```

3. **填写自评表**
   - 点击左侧菜单"填写自评表"
   - 填写必填项（至少填写一些内容）
   - 教学过程管理：输入一些文字
   - 教学质量管理：输入一些文字
   - 其他项目：可选

4. **提交表单**
   - 滚动到页面底部
   - 点击"提交表单"按钮
   - 等待提交成功提示

5. **验证提交成功**
   - [ ] 看到"提交成功！数据已上传到考评小组端"提示
   - [ ] 没有错误信息
   - [ ] 记录提交时间

#### 4.2 考评小组查看考评表

1. **退出教研室账号**
   - 点击左下角"退出登录"

2. **登录考评小组账号**
   ```
   用户名：evaluator1
   密码：password123
   选择角色：评教小组
   ```

3. **进入手动评分页面**
   - 点击左侧菜单"手动评分"

4. **查看待评分列表**
   - [ ] 能看到"计算机科学教研室"
   - [ ] 提交时间正确
   - [ ] 时间格式：`2026-02-28 HH:mm:ss`
   - [ ] 状态显示为"已提交"

5. **查看考评表详情**
   - 点击"开始评分"按钮
   - [ ] 页面正常加载
   - [ ] 显示自评表内容
   - [ ] 显示刚才填写的内容
   - [ ] 可以输入评分

## ❌ 如果测试失败

### 问题1：时间格式还是不对
**解决方法**：
1. 确认已清除浏览器缓存
2. 尝试使用无痕模式（Ctrl + Shift + N）
3. 尝试使用其他浏览器
4. 检查前端代码是否已更新

### 问题2：页面加载失败
**解决方法**：
1. 打开浏览器控制台（F12）
2. 查看 Console 标签的错误信息
3. 截图发送给开发人员
4. 尝试刷新页面

### 问题3：考评小组看不到考评表
**解决方法**：
1. 确认教研室已经提交（看到成功提示）
2. 刷新考评小组端页面
3. 检查浏览器控制台错误
4. 查看 Network 标签的 API 请求

## 🔍 调试技巧

### 查看 API 请求
1. 按 F12 打开开发者工具
2. 切换到 Network 标签
3. 刷新页面
4. 查找以下请求：
   - `/api/auth/login` - 登录
   - `/api/scoring/evaluations-for-scoring` - 获取列表
   - `/api/teaching-office/self-evaluation/{id}/submit` - 提交

### 查看响应数据
1. 点击 Network 标签中的请求
2. 切换到 Response 标签
3. 查看返回的数据
4. 确认数据格式正确
