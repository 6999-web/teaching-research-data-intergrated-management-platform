# 状态颜色优化 - 黄橙绿方案

## 修改时间
2026-02-28

## 颜色方案

### 最终方案：黄色 → 橙色 → 绿色

```
🟡 已提交 (黄色) - 已提交，等待处理
🟠 待评分 (橙色) - 需要评分
🟢 已评分 (绿色) - 已完成评分
```

## 详细配置

### 状态颜色映射

| 状态 | 标签文本 | 颜色 | Element Plus 类型 | 说明 |
|------|---------|------|------------------|------|
| draft | 草稿 | 灰色 | info | 初始状态 |
| submitted | 已提交 | 🟡 黄色 | 自定义 | 已提交，等待处理 |

| published | 已公示 | 🟢 绿色 | success | 已公示 |

## 技术实现

### 1. 状态类型函数

```typescript
const getStatusTagType = (status: string): string => {
  const types: Record<string, string> = {
    'draft': 'info',
    'submitted': '', // 使用自定义黄色样式
    'locked': '', // 使用自定义黄色样式
    'ai_scored': 'warning', // 橙色
    'manually_scored': 'success', // 绿色
    'ready_for_final': 'primary',
    'finalized': 'success',
    'published': 'success'
  }
  return types[status] || 'info'
}
```

### 2. 自定义类函数

```typescript
const getStatusClass = (status: string): string => {
  return (status === 'submitted' || status === 'locked') ? 'status-yellow' : ''
}
```

### 3. 模板使用

```vue
<el-tag 
  :type="getStatusTagType(scope.row.status)"
  :class="getStatusClass(scope.row.status)"
>
  {{ getStatusLabel(scope.row.status) }}
</el-tag>
```

### 4. 自定义黄色样式

```css
.status-yellow {
  background-color: #fef0cd !important;
  border-color: #f7d794 !important;
  color: #d68910 !important;
}

.status-yellow.el-tag {
  background-color: #fef0cd !important;
  border-color: #f7d794 !important;
  color: #d68910 !important;
}
```

## 设计理念

### 1. 渐进式颜色
- **黄色**：提醒注意，但不紧急
- **橙色**：需要处理，较为紧急
- **绿色**：已完成，状态良好

### 2. 视觉层次
- 黄色 → 橙色：从"等待"到"需要行动"
- 橙色 → 绿色：从"进行中"到"已完成"

### 3. 用户体验
- 一眼就能看出哪些需要处理
- 颜色变化符合直觉
- 绿色表示完成，给人安心感

## 扩展性

### 添加新状态
如果需要添加新状态，只需：
1. 在 `getStatusTagType` 中添加映射
2. 在 `getStatusLabel` 中添加标签文本
3. 如需自定义颜色，在 `getStatusClass` 中添加判断
4. 添加对应的 CSS 样式

### 示例：添加"审核中"状态
```typescript
// 1. 添加类型映射
'reviewing': '', // 使用自定义紫色

// 2. 添加标签文本
'reviewing': '审核中',

// 3. 添加自定义类
const getStatusClass = (status: string): string => {
  if (status === 'submitted' || status === 'locked') return 'status-yellow'
  if (status === 'reviewing') return 'status-purple'
  return ''
}

// 4. 添加 CSS
.status-purple {
  background-color: #f4f0ff !important;
  border-color: #d3adf7 !important;
  color: #722ed1 !important;
}
```
