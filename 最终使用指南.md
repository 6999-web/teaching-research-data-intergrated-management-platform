- 教学事故
- 学生投诉
- 违规违纪

1. 点击页面底部的"提交表单"按钮
2. 确认提交对话框

```
http://localhost:3000/manual-scoring
```

## ✨ 提交流程

### 用户操作
```
填写表单 → 点击"提交表单" → 确认 → 完成
```

### 系统处理
```
1. 验证表单
2. 保存数据
3. 提交并锁定
4. 触发AI评分（可选）
5. 跳转到主页
```

### 数据流转
```
教研室端提交
    ↓
后端保存（status: locked）
    ↓
AI评分（status: ai_scored）
    ↓
考评小组端列表显示
```

## 💡 提示信息

点击提交后，您会看到：

1. ℹ️ "正在提交自评表..."
2. ✅ "自评表保存成功"
3. ✅ "自评表提交成功，已锁定"
4. ❓ "是否立即触发AI评分？"
   - 立即评分 → ✅ "AI评分已启动"
   - 稍后评分 → 跳过
5. ✅ "操作完成，即将返回主页"

## ⚠️ 注意事项

### 1. 提交前
- ✅ 确保所有必填项都已填写
- ✅ 检查分数是否正确
- ✅ 确认内容无误

### 2. 提交后
- ⚠️ 表单将被锁定，无法修改
- ⚠️ 只有管理端可以解锁（打回）
- ⚠️ 请谨慎操作

### 3. 网络要求
- 📶 需要稳定的网络连接
- 📶 提交过程中不要关闭页面
- 📶 等待所有提示完成

## 🔍 验证方法

### 方法1: 查看按钮
- 页面底部只有一个蓝色的"提交表单"按钮
- 没有"保存草稿"、"预览"、"重置"按钮

### 方法2: 测试提交
- 填写表单后点击"提交表单"
- 应该直接进入提交流程
- 不会提示"请先保存"

### 方法3: 查看手动评分列表
- 访问 http://localhost:3000/manual-scoring

- 状态显示为"AI已评分"或"已锁定"

## 🐛 常见问题

### Q1: 刷新后还是看到4个按钮？
**A**: 清除浏览器缓存
```
1. 按 Ctrl + Shift + Delete
2. 清除缓存
3. 重新访问页面
```

### Q2: 点击提交没有反应？
**A**: 检查表单是否填写完整
- 查看浏览器控制台（F12）
- 检查是否有红色错误提示
- 确保所有必填项都已填写

### Q3: 提交后在手动评分页面看不到？
**A**: 等待AI评分完成
- AI评分需要几秒到几分钟
- 刷新手动评分页面
- 检查自评表状态是否为 `ai_scored`

### Q4: 提示网络错误？
**A**: 检查服务状态
- 后端: http://localhost:8000/docs
- 前端: http://localhost:3000
- 数据库: localhost:3306
