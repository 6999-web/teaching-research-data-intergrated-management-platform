
### 手动修改状态

```javascript
// 模拟考评完成
evaluationCompleted.value = true

// 模拟已提交
improvementSubmitted.value = true

// 重置提交状态
improvementSubmitted.value = false
```

## 🐛 常见问题

### Q1: 看不到"下学期改进措施"菜单
**A**: 确保已登录教研室账号（director1）

### Q2: 一直显示"考评尚未完成"
**A**: 修改代码中的 `evaluationCompleted` 为 `true`，或者等待后端 API 返回完成状态

### Q3: 提交后还能修改
**A**: 检查 `improvementSubmitted` 状态是否正确设置为 `true`

### Q4: 提交失败
**A**: 
1. 检查所有必填字段是否已填写
2. 检查网络连接
3. 查看浏览器控制台的错误信息
