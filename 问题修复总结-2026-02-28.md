### 1. 手动评分页面时间显示格式 ✓

#### 问题描述
考评小组端的手动评分页面显示的时间不是真实时间格式，使用了本地化格式。

#### 修复内容
**文件**：`frontend/src/views/ManualScoring.vue`

**修改前**：
```typescript
const formatDate = (dateStr: string): string => {
  const date = new Date(dateStr)
  return date.toLocaleString('zh-CN', {
    year: 'numeric',
    month: '2-digit',
    day: '2-digit',
    hour: '2-digit',
    minute: '2-digit'
  })
}
```
显示效果：`2026/2/28 下午2:30`

**修改后**：
```typescript
const formatDate = (dateStr: string): string => {
  if (!dateStr) return '-'
  const date = new Date(dateStr)
  const year = date.getFullYear()
  const month = String(date.getMonth() + 1).padStart(2, '0')
  const day = String(date.getDate()).padStart(2, '0')
  const hours = String(date.getHours()).padStart(2, '0')
  const minutes = String(date.getMinutes()).padStart(2, '0')
  const seconds = String(date.getSeconds()).padStart(2, '0')
  
  return `${year}-${month}-${day} ${hours}:${minutes}:${seconds}`
}
```
显示效果：`2026-02-28 14:30:45`

### 2. 手动评分页面加载问题 ✓

#### 问题描述
手动评分页面无法加载，浏览器控制台显示错误：
```
Uncaught (in promise) TypeError: Cannot read properties of undefined (reading 'score')
```

#### 问题原因
在模板中使用 `formData.scores[itemKey]` 时，如果该 `itemKey` 还没有被初始化，就会导致访问 undefined 对象的属性错误。

#### 修复内容
**文件**：`frontend/src/components/ManualScoringForm.vue`

**修改1：添加安全检查**
```vue
<!-- 修改前 -->
<div class="scoring-area">
  <el-input-number v-model="formData.scores[itemKey].score" />
</div>

<!-- 修改后 -->
<div v-if="formData.scores[itemKey]" class="scoring-area">
  <el-input-number v-model="formData.scores[itemKey].score" />
</div>
```

**修改2：优化初始化逻辑**
```typescript
// 在加载自评表后立即初始化表单数据
const loadSelfEvaluation = async () => {
  const response = await selfEvaluationApi.get(props.evaluationId)
  selfEvaluationContent.value = response.data.content
  
  // 立即初始化表单数据
  initializeFormData()
}
```

**修改3：简化初始化函数**
```typescript
const initializeFormData = () => {
  if (!selfEvaluationContent.value) return
  
  Object.keys(selfEvaluationContent.value).forEach(sectionKey => {
    const section = selfEvaluationContent.value[sectionKey]
    if (typeof section === 'object' && section !== null) {
      Object.keys(section).forEach(itemKey => {
        // 直接赋值，确保响应式
        formData.scores[itemKey] = {
          score: 0,
          comment: ''
        }
      })
    }
  })
}
```
