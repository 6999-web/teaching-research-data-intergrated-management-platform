## 📝 使用说明

### 用户操作流程

1. 填写字段内容
2. 在右侧附件区域上传相关文件（可选）
3. 可以为每个字段单独上传附件
4. 提交表单时，所有附件会一起上传

### 开发者说明

#### 添加新字段的附件上传

如果需要为新字段添加附件上传功能，按以下步骤操作：

1. **添加文件列表变量**
```typescript
const newFieldFiles = ref<UploadUserFile[]>([])
```

2. **在模板中添加附件上传区域**
```vue
<div class="content-with-attachment">
  <div class="content-section">
    <!-- 原有的输入框 -->
  </div>
  
  <div class="attachment-section">
    <el-card class="mini-attachment-card" shadow="hover">
      <!-- 附件上传组件 -->
      <el-upload
        :file-list="newFieldFiles"
        :on-change="(file, fileList) => handleInlineFileChange(file, fileList, 'newFieldKey')"
        :on-remove="(file, fileList) => handleInlineFileRemove(file, fileList, 'newFieldKey')"
        <!-- 其他配置 -->
      />
    </el-card>
  </div>
</div>
```

3. **更新handleInlineFileChange方法**
在fieldNameMap中添加新字段的映射，并在switch语句中添加case。

4. **更新handleSubmit方法**
在allAttachments数组中添加新字段的文件列表。

## 🔧 技术细节

### 文件验证

1. **大小验证**：单个文件不超过 50MB
2. **格式验证**：只允许 pdf, doc, docx, xls, xlsx, jpg, jpeg, png
3. **数量限制**：每个字段最多 5 个文件

### 错误处理

- 文件过大：显示错误提示并自动移除
- 格式不支持：显示错误提示并自动移除
- 超出数量限制：显示警告提示

### 用户反馈

- ✅ 添加文件成功提示
- ℹ️ 移除文件信息提示
- ⚠️ 超出限制警告提示
- ❌ 验证失败错误提示
