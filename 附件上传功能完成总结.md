## 🔧 技术实现

### 1. 文件上传组件
使用 Element Plus 的 `el-upload` 组件：
- `auto-upload="false"` - 不自动上传，等待表单提交
- `multiple="true"` - 支持多文件选择
- `drag` - 支持拖拽上传
- `limit="5"` - 限制最多5个文件

### 2. 文件验证逻辑
```typescript
// 大小验证
if (file.size > 50 * 1024 * 1024) {
  ElMessage.error('文件过大')
  return
}

// 格式验证
const allowedTypes = ['pdf', 'doc', 'docx', 'xls', 'xlsx', 'jpg', 'jpeg', 'png']
if (!allowedTypes.includes(fileExt)) {
  ElMessage.error('格式不支持')
  return
}
```

### 3. 文件元数据
每个文件都会添加元数据，便于后续处理：
```typescript
{
  indicator: 'teachingProcessManagement',  // 字段标识
  fieldName: '教学过程管理',               // 字段中文名
  name: 'document.pdf',                    // 文件名
  size: 1024000,                           // 文件大小
  // ... 其他属性
}
```

### 4. 提交时合并附件
```typescript
const allAttachments = [
  ...attachmentFileList.value,      // 底部统一上传
  ...teachingProcessFiles.value,    // 字段1的附件
  ...teachingQualityFiles.value,    // 字段2的附件
  // ... 其他字段的附件
]
```
