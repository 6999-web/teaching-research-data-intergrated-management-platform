## 🐛 常见问题

### Q1: 前端无法启动
**A**: 检查是否安装了依赖
```bash
cd frontend
npm install
npm run dev
```

### Q2: 后端无法启动
**A**: 检查Python环境和依赖
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### Q3: 登录失败
**A**: 检查后端是否正常运行，访问 http://localhost:8000/docs 查看API文档

### Q4: 附件上传按钮不显示
**A**: 清除浏览器缓存，强制刷新页面（Ctrl+Shift+R）

### Q5: 改进措施显示"考评未完成"
**A**: 这是正常的，需要修改代码设置 `evaluationCompleted = true` 才能测试填写功能
