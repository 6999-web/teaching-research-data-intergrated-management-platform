## 🔧 如何切换到其他数据库

### 切换到 SQLite (开发测试)

1. 修改 `backend/.env`:
```env
USE_SQLITE=true
```

2. 重启后端服务

### 切换到远程 MySQL

1. 修改 `backend/.env`:
```env
MYSQL_SERVER=远程服务器IP
MYSQL_USER=用户名
MYSQL_PASSWORD=密码
MYSQL_DB=数据库名
MYSQL_PORT=3306
```

2. 重启后端服务
