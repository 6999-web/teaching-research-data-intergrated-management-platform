# 附件上传按钮优化

## 修改内容

将所有评分项的附件上传区域从复杂的拖拽卡片改成简单的按钮形式。

## 修改前

每个评分项右侧有一个大的卡片区域：
- 包含拖拽上传区域
- 显示"拖拽或点击上传"
- 占用较大空间

## 修改后

每个评分项右侧只有一个简洁的按钮：
- 点击按钮直接弹出文件选择对话框
- 按钮显示已上传文件数量：`附件上传 (2)`
- 已上传的文件显示在按钮下方的列表中
- 每个文件可以单独删除

## 新功能特点

### 1. 简洁的按钮
```vue

  附件上传 ({{ teachingProcessFiles.length }})
</el-button>
```

### 2. 文件列表显示
- 显示文件图标
- 显示文件名
- 提供删除按钮

### 3. 提示信息
```
支持 PDF、Word、Excel、图片
单个≤50MB，最多5个
```
