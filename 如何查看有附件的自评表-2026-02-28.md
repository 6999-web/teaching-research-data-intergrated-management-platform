## 如何查看有附件的自评表

### 步骤1：返回自评表列表

在手动评分页面，点击"返回列表"按钮

### 步骤2：选择有附件的自评表

在自评表列表中，找到提交时间为 **2026-02-28 17:33:34** 的那一条（这是第2个）

### 步骤3：开始评分

点击"开始评分"按钮

### 步骤4：查看附件

现在你应该能看到3个附件：
- test_teaching_quality.txt
- test_course_assessment.txt  
- test_teaching_process.txt

## 为什么有些自评表没有附件？

可能的原因：
1. 教研室端提交时没有上传附件
2. 这些是早期测试数据，当时附件功能还没有完善
3. 教研室端选择不上传附件（附件是可选的）

## 如何创建有附件的测试数据

### 方式1：使用自动化测试脚本

```bash
cd teaching-research-data-intergrated-management-platform-main
python test_attachment_complete_flow.py
```

这会自动创建一个新的自评表并上传3个附件。

### 方式2：手动在教研室端上传

1. 登录教研室端（director1 / password123）
2. 创建新的自评表
3. 在每个评分项下上传附件
4. 提交自评表
