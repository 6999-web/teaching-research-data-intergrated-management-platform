# 附件上传下载问题快速修复

## 问题
- 教研室端上传的文件，考评小组端无法显示
- 考评小组端无法下载文件

## 快速修复

### 1. 重启后端服务（最重要！）

后端服务可能已停止，需要重新启动：

```powershell
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 清除浏览器缓存

```
按 Ctrl + Shift + Delete
清除缓存和图片
```

### 3. 重新测试

#### 教研室端：
1. 登录：director1 / password123
2. 填写自评表
3. 上传附件
4. 提交表单

#### 考评小组端：
1. 登录：evaluator1 / password123
2. 进入手动评分
3. 选择教研室
4. 查看附件是否显示

## 检查清单

- [ ] 后端服务运行在 http://localhost:8000
- [ ] 前端服务运行在 http://localhost:3000
- [ ] 浏览器缓存已清除
- [ ] 教研室端可以上传附件
- [ ] 考评小组端可以看到附件
- [ ] 考评小组端可以下载附件

## 如果仍有问题

打开浏览器控制台（F12），查看：
1. Console 标签 - 查看错误信息
2. Network 标签 - 查看 API 请求
3. 找到 `/api/teaching-office/attachments/` 请求
4. 查看状态码和响应

常见错误：
- 401: 需要重新登录
- 404: 自评表不存在
- 500: 后端服务错误
